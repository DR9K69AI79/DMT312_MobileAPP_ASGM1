# 学习资源库页（Screen 5）设计

## 页面概述
学习资源库页面是用户获取健身知识和营养科普的内容中心，提供离线Markdown文章阅读和收藏管理功能。页面采用卡片式布局，强调内容展示和阅读体验，帮助用户系统学习健身相关知识。

## 卡片布局

### 1. 顶部Dashboard卡片（收起状态）
- **高度**：64dp
- **宽度**：全屏宽
- **背景色**：主色调（#1E88E5）
- **内容**：同其他页面的顶部Dashboard卡片
- **交互**：点击任意位置返回Dashboard首页

### 2. 资源分类标签卡片
- **尺寸**：高度60dp，宽度为屏幕宽度减去左右边距（24dp）
- **位置**：顶部Dashboard卡片下方16dp
- **背景色**：白色（#FFFFFF）
- **内容布局**：
  - 水平滚动的分类标签列表
  - 标签包含：
    - "全部"（默认选中）
    - "训练技巧"
    - "营养科普"
    - "健身计划"
    - "收藏"
  - 每个标签：
    - 圆角胶囊形（高度36dp，宽度根据文字长度自适应）
    - 文字（14sp，选中时白色，未选中时主色调，Roboto Medium）
    - 背景（选中时主色调，未选中时白色带主色调边框）
- **交互**：
  - 点击标签：切换到对应分类的文章列表
  - 水平滑动：滚动查看更多分类标签

### 3. 搜索卡片
- **尺寸**：高度60dp，宽度为屏幕宽度减去左右边距（24dp）
- **位置**：资源分类标签卡片下方12dp
- **背景色**：白色（#FFFFFF）
- **内容布局**：
  - 搜索框（高度48dp）：
    - 左侧：搜索图标
    - 中间：输入提示"搜索文章..."（14sp，浅灰色，Roboto Regular）
    - 右侧：清除按钮（仅在有输入时显示）
- **交互**：
  - 点击搜索框：显示虚拟键盘
  - 输入搜索词：实时过滤文章列表
  - 点击清除按钮：清空搜索内容，恢复完整文章列表

### 4. 推荐文章卡片
- **尺寸**：高度220dp，宽度为屏幕宽度减去左右边距（24dp）
- **位置**：搜索卡片下方12dp
- **背景色**：白色（#FFFFFF）
- **内容布局**：
  - 顶部：标题"推荐阅读"（16sp，黑色，Roboto Medium）
  - 中部：水平滚动的文章卡片列表
    - 每个文章卡片（宽度160dp，高度160dp）：
      - 顶部：文章封面图（高度90dp）
      - 中部：文章标题（14sp，黑色，Roboto Medium，最多2行）
      - 底部：分类标签（12sp，主色调，Roboto Regular）
  - 底部：分页指示点（当前页白色，其他页灰色）
- **交互**：
  - 水平滑动：浏览更多推荐文章
  - 点击文章卡片：打开文章阅读页面

### 5. 文章列表卡片
- **尺寸**：高度自适应（根据文章数量），宽度为屏幕宽度减去左右边距（24dp）
- **位置**：推荐文章卡片下方12dp
- **背景色**：白色（#FFFFFF）
- **内容布局**：
  - 顶部：当前分类标题（16sp，黑色，Roboto Medium）+ 排序下拉菜单（14sp，灰色，Roboto Regular）
  - 文章列表：
    - 每个文章项（高度100dp）：
      - 左侧：文章缩略图（正方形，80dp x 80dp）
      - 右侧：
        - 文章标题（16sp，黑色，Roboto Medium，最多2行）
        - 简短描述（14sp，灰色，Roboto Regular，最多2行）
        - 底部：分类标签 + 阅读时间估计（12sp，浅灰色，Roboto Regular）
      - 右上角：收藏按钮（星形图标，收藏时填充，未收藏时轮廓）
- **交互**：
  - 点击文章项：打开文章阅读页面
  - 点击收藏按钮：切换文章收藏状态
  - 点击排序菜单：选择排序方式（最新/最热/字母顺序）
  - 上拉滚动：加载更多文章

### 6. 阅读进度卡片
- **尺寸**：高度自适应（最多显示3项），宽度为屏幕宽度减去左右边距（24dp）
- **位置**：文章列表卡片下方12dp（仅在有阅读历史时显示）
- **背景色**：白色（#FFFFFF）
- **内容布局**：
  - 顶部：标题"继续阅读"（16sp，黑色，Roboto Medium）+ "查看全部"文字按钮（14sp，主色调）
  - 阅读历史列表：
    - 每个历史项（高度80dp）：
      - 左侧：文章缩略图（圆角矩形，60dp x 60dp）
      - 中间：
        - 文章标题（14sp，黑色，Roboto Medium，单行）
        - 上次阅读时间（12sp，灰色，Roboto Regular）
      - 右侧：阅读进度条（垂直，高度40dp）+ 进度百分比（12sp，主色调，Roboto Regular）
- **交互**：
  - 点击历史项：继续阅读文章，从上次位置开始
  - 点击"查看全部"：显示完整阅读历史列表

## 文章阅读页面设计

### 1. 阅读页面顶部栏
- **高度**：56dp
- **宽度**：全屏宽
- **背景色**：白色（#FFFFFF）
- **内容布局**：
  - 左侧：返回按钮
  - 中间：文章标题（16sp，黑色，Roboto Medium，单行，超出部分省略）
  - 右侧：收藏按钮 + 更多选项按钮（三点菜单）
- **交互**：
  - 点击返回：返回资源库页面
  - 点击收藏：切换文章收藏状态
  - 点击更多选项：显示操作菜单（分享/调整字体大小/夜间模式）

### 2. 文章内容区域
- **位置**：顶部栏下方
- **背景色**：白色（#FFFFFF）
- **内容布局**：
  - 顶部：文章封面图（如有）
  - 标题：文章标题（24sp，黑色，Roboto Bold）
  - 元信息：分类 + 阅读时间（14sp，灰色，Roboto Regular）
  - 正文：Markdown渲染内容
    - 段落文字（16sp，黑色，Roboto Regular，行高1.5）
    - 标题层级（h1: 22sp, h2: 20sp, h3: 18sp，Roboto Medium）
    - 列表项（16sp，黑色，Roboto Regular，带适当缩进和项目符号）
    - 图片（全宽显示，保持原始比例）
    - 表格（水平滚动支持）
    - 代码块（等宽字体，浅灰色背景）
- **交互**：
  - 上下滑动：阅读文章内容
  - 双击段落：选择文本
  - 长按文本：显示复制/分享选项
  - 点击图片：全屏查看图片

### 3. 底部操作栏
- **高度**：56dp
- **宽度**：全屏宽
- **背景色**：白色（#FFFFFF）
- **内容布局**：
  - 左侧：阅读进度百分比（14sp，灰色，Roboto Regular）
  - 中间：水平进度条（高度4dp，主色调）
  - 右侧：字体大小调整按钮（A- / A+）
- **交互**：
  - 拖动进度条：快速跳转到文章指定位置
  - 点击字体大小按钮：增大/减小文章字体

## 页面交互流程

### 1. 文章浏览流程
- 进入资源库页面：
  - 默认显示"全部"分类的文章列表
  - 推荐文章卡片自动轮播（每5秒切换一次）
- 分类筛选：
  - 点击分类标签：切换到对应分类
  - 文章列表平滑刷新为该分类内容
- 搜索筛选：
  - 在搜索框输入关键词
  - 文章列表实时过滤，仅显示匹配结果
  - 无匹配结果时显示"未找到相关文章"提示

### 2. 文章阅读流程
- 点击文章项：
  - 打开文章阅读页面（全屏）
  - 页面切换动画：从下向上滑入
- 阅读操作：
  - 上下滑动阅读内容
  - 底部进度条实时更新
  - 系统自动记录阅读进度
- 退出阅读：
  - 点击返回按钮或使用系统返回手势
  - 页面切换动画：从上向下滑出
  - 返回资源库页面，保持原分类和滚动位置

### 3. 收藏管理流程
- 收藏文章：
  - 在文章列表或阅读页面点击收藏按钮
  - 按钮动画：从轮廓变为填充状态
  - 显示轻量级成功提示
- 查看收藏：
  - 点击"收藏"分类标签
  - 文章列表更新为已收藏文章
- 取消收藏：
  - 再次点击已收藏文章的收藏按钮
  - 按钮动画：从填充变为轮廓状态
  - 如当前在"收藏"分类，文章项平滑移除

## 特殊交互功能

### 1. 阅读模式切换
- 点击阅读页面更多选项中的"夜间模式"：
  - 背景色变为深色（#121212）
  - 文字颜色变为浅色（#E0E0E0）
  - 减少蓝光，适合夜间阅读
- 系统会记住用户偏好，下次阅读自动应用

### 2. 文章分享
- 点击分享按钮：
  - 显示系统分享菜单
  - 可分享文章标题和应用下载链接
  - 或生成文章截图分享

### 3. 阅读进度同步
- 退出文章时自动保存阅读进度
- 下次打开同一文章时，自动滚动到上次阅读位置
- "继续阅读"卡片显示最近阅读的文章

## 响应式调整
- **小屏幕设备**（<5英寸）：
  - 推荐文章卡片宽度减小至140dp
  - 阅读页面字体大小整体减小2sp
  
- **大屏幕设备**（>6英寸）：
  - 推荐文章一屏显示3个
  - 阅读页面左右增加边距至32dp，提升阅读体验

## 状态与反馈
- **首次使用**：显示引导提示，介绍资源库功能
- **加载状态**：文章内容加载时显示骨架屏
- **离线状态**：所有文章支持离线阅读，显示"离线模式"标识
- **收藏成功**：显示轻量级成功提示
