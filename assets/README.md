# 文章封面图片

这个目录包含文章的封面图片文件。

支持的图片格式：
- PNG (.png)
- JPEG (.jpg, .jpeg)
- WebP (.webp)

文件命名规范：
- muscle_gain_cover.jpg - 增肌文章封面
- fat_burn_cover.jpg - 燃脂文章封面
- nutrition_cover.jpg - 营养文章封面
- recovery_cover.jpg - 恢复文章封面

在Front Matter中使用方式：
```yaml
coverUrl: "assets/images/muscle_gain_cover.jpg"
```
